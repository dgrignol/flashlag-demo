===============================

This project runs a simple browser demo of the Flash-Lag Illusion
implemented in React.

------------------------------------------------------------
How to run the project
------------------------------------------------------------

1. Open a terminal.

2. Go into the project folder:
   cd /Users/damiano/flashlag-demo/src

3. Start the development server:
   npm run dev

4. Open the URL shown in the terminal (for example http://localhost:5173).
   The game will be available in your browser.

------------------------------------------------------------
How to stop the project
------------------------------------------------------------

- In the terminal where the server is running, press CTRL+C
  (or close the terminal window).

------------------------------------------------------------
How to run it again later
------------------------------------------------------------

1. Open a terminal.
2. Go into the project folder:
   cd /Users/damiano/flashlag-demo/src
3. Run:
   npm run dev
4. Open the URL shown in the terminal.

------------------------------------------------------------
Optional: Create a static build
------------------------------------------------------------

If you want to generate a standalone build you can host anywhere:

1. Open a terminal and go into the project folder:
   cd /Users/damiano/flashlag-demo/src

2. Run:
   npm run build

   This creates a "dist" folder with plain HTML/JS/CSS.

3. To preview the built version locally:
   npm run preview
